# tailwindcss-postcss-plugin-repro

```sh
git clone https://github.com/t-mart/tailwindcss-postcss-plugin-repro.git
cd tailwindcss-postcss-plugin-repro
npm install
```

## Working (3.0.24)

```sh
npm install tailwindcss@3.0.24
npm run build
```

## Not Working (3.1.0)

```sh
npm install tailwindcss@3.1.0
npm run build
```
